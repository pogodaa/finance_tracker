Проект: "Персональный финансовый менеджер (Finance Tracker)"

🔥 Концепт проекта: Это веб-приложение для учёта личных доходов и расходов с аналитикой, категориями и отчётами.

📌 Основные функции
1. Учёт операций
    • Добавление доходов/расходов:
        ◦ Сумма, дата, категория, описание.
        ◦ Прикрепление файлов (чеки, скриншоты).
    • Категории:
        ◦ Еда, транспорт, развлечения, здоровье и т. д.
        ◦ Возможность создавать свои категории.
2. Аналитика
    • Графики и диаграммы:
        ◦ Круговая диаграмма «На что трачу больше всего».
        ◦ График динамики расходов по месяцам.
    • Фильтры:
        ◦ По дате, категории, типу (доход/расход).
3. Отчёты
    • Экспорт в CSV/Excel.
    • PDF-отчёт за месяц.
4. Пользователи и авторизация
    • Регистрация/вход (email + пароль).


🛠 Стек технологий
Backend: FastAPI + SQLAlchemy ORM  
База данных: PostgreSQL (ElephantSQL) + Redis (Redis Cloud)  
API: REST + Swagger  
Авторизация: JWT 
Фронтенд: Bootstrap 5 (JS)
Деплой: Docker + GitHub Actions + Render  
Тестирование: Pytest + Postman  
Дополнительно: Telegram Bot
